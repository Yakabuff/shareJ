# shareJ
Java ShareX clone

## Installation

```java -jar shareJ_v0.0.1.jar```

## Features

- Capture fullscreen
- Capture selection
- Stores screenshots on disk
- Puts image on clipboard
- Minimize to tray
- SQLite database storing image history
